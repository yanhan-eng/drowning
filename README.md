# 智能监控：基于机器视觉的防溺水预警系统
1. 关于本项目
   
溺水是全球非蓄意伤害死亡的重要原因之一，传统的水域安全高度依赖人工巡视，存在视觉盲区大、容易疲劳、响应不及时等痛点。

本项目旨在开发一套**“全天候、高时效、零接触”**的智能视频监控预警系统。系统通过引入先进的计算机视觉技术，能够对监控视频流中的人员进行实时追踪，并结合人体行为学特征（如水下挣扎、垂直位移变化、水平移动停滞等），智能研判水域内是否存在溺水险情。一旦发现异常，系统将立即在屏幕上发出警报，为黄金救援赢取宝贵时间。

6.📈 测试结果与系统表现

经过多次模拟场景与真实视频的交叉测试：

精准度识别： 系统能够对水中的目标人员实现持续、稳定的动态追踪。

误报率控制： 得益于系统独创的“动作时间窗口与位移比例”研判逻辑，系统能够有效区分“正常的短暂潜水/换气”与“真实的溺水挣扎”，大幅降低了传统系统的误报率。

实时响应能力： 系统从捕捉到溺水动作到触发警报。在标准测试集下，视频画面流畅，无明显卡顿，完全满足现实水域监控对“时效性”的严苛要求。

可调节性： 针对不同视角的监控探头，系统允许操作人员在左侧控制台灵活调整“动作判定窗口”和“灵敏度”，使其具备极强的环境适应能力。
